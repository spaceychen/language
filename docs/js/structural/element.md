# 基本的组织元素

语法元素及其组织形态

| 元素   | 物理形态   | 静态                                | 动态                                         |
| ------ | ---------- | ----------------------------------- | -------------------------------------------- |
| 标识符 |            | var/let/const<br>函数声明<br>类声明 | 非严格模式下的 var<br>非严格模式下的函数声明 |
| 表达式 | 模板字符串 | 值<br>箭头函数体                    | 通过 eval 执行表达式语句来实现               |
| 语句   | .js 文件   | 块与块级作用域                      | eval()                                       |
| 模块   | .mjs 文件  | import/export<br>require()          | import().then()                              |
